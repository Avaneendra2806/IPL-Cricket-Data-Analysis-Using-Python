# 🏏 IPL Cricket Data Analysis using Python
## 📖 Overview
This project presents an end-to-end Exploratory Data Analysis (EDA) of the Indian Premier League (IPL) dataset using Python. The objective is to analyze team performance, player statistics, match outcomes, and venue trends by applying data cleaning, feature engineering, exploratory data analysis, and data visualization techniques.

The project demonstrates practical data analytics skills using Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn to extract meaningful insights from IPL match and ball-by-ball data.
---
## 🎯 Objectives
- Analyze overall team performance across IPL seasons.
- Evaluate batting and bowling performances.
- Explore player statistics and match-winning performances.
- Analyze toss decisions and their impact on match outcomes.
- Study venue-wise and city-wise match trends.
- Perform feature engineering to improve data analysis.
- Generate business insights through exploratory data analysis.
